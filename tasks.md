# build-tasks

1. コマンド実行
   `docker-compose build`

1. コマンド実行
   `docker-compose run --rm backend rails db:create`

1. コマンド実行、vue プロジェクト作成
   `docker-compose run --rm frontend yarn install`

1. ビルド
   `docker-compose build`

1. 立ち上げ
   `docker-compose up`

1. 接続テスト
   rails のページ`localhost:3000`
   vue のページ`localhost:8080`
